

##### Borrar ls, arreglarlo
$ yum provides ls
$ rm /bin/ls
$ yum reinstall -y coreutils

##### Que paquete me instala el /bin/bash?
$ yum provides /bin/bash

##### En el histórico del yum, cuales son los últimos paquetes instalados en el sistema?
$ yum history info <último ID> 

##### Instalar el grupo "soporte de Esperanto"
$ yum groupinstall "Esperanto Support"

##### Quien me instala la libreria /usr/lib64/libsoprano.so.4 ?
$ yum provides /usr/lib64/libsoprano.so.4
soprano-2.3.1-1.2.el6.x86_64

##### Instala el repositorio EPEL
$ rpm -ivh http://download.fedoraproject.org/pub/epel/7/x86_64/e/epel-release-7-8.noarch.rpm

##### Instala el comando unarj (EPEL)
$ yum provides *bin*unarj
$ yum install arj

##### Quien me instala la libreria /usr/lib64/libcaca.a ? (EPEL)
$ yum provides /usr/lib64/libcaca.a
libcaca-devel-0.99-0.9.beta16.el6.x86_64

##### Bajarse el paquete p7zip sin instalarlo (EPEL)
$ yum install yum-plugin-downloadonly
$ yum install p7zip --downloadonly --downloaddir=.

##### Instalar soporte para NTFS (EPEL)
$ yum install ntfs-3g
