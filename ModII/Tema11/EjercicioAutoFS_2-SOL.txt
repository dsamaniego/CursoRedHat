EJERCICIO AUTOFS 2  - SOLUCION
-------------------------------

Hacer reset de serverX antes de empezar.

En /docsMarta de serverX, montar el recurso /documentos de server0 con automonter.

En /docsRHEL de serverX, montar el recurso /content/ de server0 con automonter.


SOLUCIÃ“N
=========

1) Identifico los recursos que exporta server0:

   mkdir /mnt/server0
   mount -t nfs server0:/ /mnt/server0
   ls /mnt/server0; yum install tree -y;tree /mnt/server0

 Veo que exporta /documentos y que dentro hay un/os fichero/s -> montaje directo
  y veo que exporta /content y dentro hay subdirectorios -> montaje indirecto

  umount /mnt/server0;rmdir /mnt/server0

 2) Instalo paquete autofs en serverX:

   yum install -y autofs 

3) Creo los puntos de montaje que voy a necesitar en serverX:

  mkdir /{docsMarta,docsRHEL}
  
4) Montaje directo de /docsMarta:
 Creo fichero principal /etc/auto.master.d/marta.autofs con el contenido:
      /-  /etc/auto.marta

4) Creo el fichero con los detalles del montaje: /etc/auto.marta con el contenido:
    /docsMarta   -rw,sync   server0:/documentos

5) Montaje indirecto de /docsRHEL: 
  Creo fichero principal /etc/auto.master.d/docsrhel.autofs con el contenido:
/docsRHEL /etc/auto.docsrhel

4) Creo el fichero con los detalles del montaje: /etc/auto.docsrhel con el contenido:
    * -rw,sync server0:/content/&

6) Arranco y activo el servicio autofs:

   systemctl start autofs && systemctl enable autofs

7) Compruebo si veo el contenido

   ls /docs/RHEL/Sistema;ls /docs/SELinux;ls /docs/Virtualizacion
