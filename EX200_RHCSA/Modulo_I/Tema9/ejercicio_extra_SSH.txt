EJERCICIO EXTRA SSH
--------------------
1. Crear en serverX un usuario llamado "admin2" con password "admin2" que pertenezca al grupo "sistemas"

2. Hacer que desde desktopX el usuario "student" se conecte a serverX con el usuario "admin2" usando autenticación basada en claves
  NOTA: En desktopX NO debe existir el usuario "admin2"

3. Crear en serverX un directorio colaborativo /home/sistemas para los usuarios del grupo "sistemas" donde sólo cada
   usuario pueda borrar sus ficheros.

4. Conectar desde desktopX a serverX con el usuario "admin2" y crear el fichero "hola.txt" en el directorio colaborativo
    
5. En desktopX, como usuario "student", crear un fichero "usuarios.txt" que contenga las 7 ultimas líneas del fichero /etc/passwd de serverX.
  Copiar este fichero con rsync en el /tmp de serverX usando el usuario "admin2"
